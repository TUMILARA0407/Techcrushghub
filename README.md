Techcrushghub — simple static starter

Files:

- index.html — main webpage
- styles.css — site styles

To preview:

1. Open `index.html` in your browser (double-click or use "Open File").
2. For a local server, run: `python -m http.server` in this folder and open http://localhost:8000
